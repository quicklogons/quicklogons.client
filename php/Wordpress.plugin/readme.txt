Fast and easy get users logged in on your web-site with bunch of OAuth providers like Google, Facebook, Twitter and more.

== Description ==

We hope you want to increase user's traffic in your blog and we want to help you reach this goal.
You may know that users are registering on web-sites in case only they really need something, 
but lots surfers just close page when see that registration is required. Somebody just lazy, 
somebody doesn't want to spend time to register... or cares about security and personal data.
So, how to engage user to login? Really, easy - just let users login with one of social network login, 
like Google, Facebook, Twitter and more.
Earlier you must have developers skills to enable on your site OAuth, but now we care about it for you.
All that you need - follow few simple steps:
    1. Install and enable this plugin, go to settings page
    2. Login to www.quicklogons.com and enter profile
    3. Add site settings, put callback url and save
    4. Get site key and secret, put them to plugin settings and save
    5. Enjoy!
Those steps are enought to start. It would take only few minutes, 
but you will give surfers more freedom and comfort and as result increase traffic.

== Installation ==

1. Upload `quicklogons` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Quicklogons plugin settings and copy callback url
3. Enter http://www.quicklogons.com/ and login to your profile
4. Add new site and put copied callback url in appropriaate field
5. Save 
6. Site key and secret were generated for you, copy them
7. Back to Quicklogons plugin settings, put site key and secret to appropriate fields
8. Save Changes
9. Enjoy!

== Frequently Asked Questions ==

= Can I disable login to some of providers? =

Yes. You can do that on www.quicklogons.com in your profile > site settings > OAuth providers.

= Can I provide custom providers configuration? =

Yes. And we so much appreciate it. Because it makes more trust for login users. Refer documentation to get more information.

= Is it free? =

Yes, but not yet:) Actually, we have personal plan that is free, but your users after successful login will see 5 sec AD.

= I expirienced with issue. What to do? =

Please contact us by mail support@quicklogons.com and decribe issue. We will do our best to fix it asap.
